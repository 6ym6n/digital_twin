# Installation Guide

This project has a Windows-first quickstart. For a step-by-step setup, use:

- Project quick start: [README](../README.md)
- PDF installation guide: [guide_installation.pdf](guide_installation.pdf)

## Prerequisites

- Python 3.9+
- Node.js 16+
- MQTT broker (Mosquitto)
- MATLAB R2020b+ (required for simulation)

## Notes

- The knowledge-base PDF must be present at `data/grundfos-cr-pump-troubleshooting.pdf`.
- The vector DB is generated in `chroma_db/` (delete it to rebuild).
